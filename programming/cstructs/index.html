Structures in C

A struct in C is a composite data type. This means that it acts as a containter and gropups variables under one name in a block of memory. Some languages call it "record", like Erlang. Structs aren't objects, they are different, their fields are public and they can't have methods.

Declaration

Use following syntax for declaring structs: 

struct name {
    type member1;
    type member2;
    /* you can declare as many as you want, but the size should be know to compiler */
};
